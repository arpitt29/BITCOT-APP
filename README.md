# PHP Application Deployment using Jenkins & Docker on AWS EC2

## Overview
This project demonstrates CI/CD deployment of a PHP web app connected to Amazon RDS using Jenkins pipelines.

## Setup Steps
1. Install Docker and Jenkins on your EC2 instance.
2. Clone this repository.
3. Update your `config.php` with RDS credentials.
4. Create a Jenkins Pipeline job pointing to this repo.
5. Run the pipeline.
